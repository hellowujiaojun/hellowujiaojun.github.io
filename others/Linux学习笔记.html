文件类型与权限

  • 第一个字符：
    • d：dir目录
    • -：文件
    • l: link链接
    • b: 设备文件里面的可供存储的周边设备（可按块随机读写的设备）
    • c：设备文件里面的串行端口设备，如键盘，鼠标
  • 第一个rwx：文件拥有者可具备的权限
    • r: read
    • w: write
    • x: execute
    • -: 无该权限
  • 第二个rwx：加入此用户组账号的权限
  • 第三个rwx：非本人且没有加入本用户组的其他账号的权限
    对于目录，若r–，则无法进入。若r-x，则可进入，但无法写入。
    文件日期
    文件日期为简写，若查看全部时间: ls -l –full-time
    文件名
    若文件名前有.，则为隐藏文件。
    更多ls操作，可查看ls –help 或man ls或info ls

修改文件的属性与权限
chgrp:修改文件所属用户组
chown:修改文件拥有者
chmod:修改文件的权限，SUID,SGID,SBIT等特性

chgrp:
change group.
chgrp [-R] dirname/filename
-R：recursive（递归），即修改目录子文件和子目录。
如：chgrp users filename

chown:
change owner

Vim

vim的模式

  • command mode
  • insert mode
  • command-line mode

文件打开与创建：

若文件存在，则打开文件。若文件不存在，则创建文件。

insert mode

command mode –> insert mode

command full name illustration
i与I insert mode 进入insert mode。i为【从当前光标处插入】。I为【从所在行第一个非空格字符处插入】
a与A append at eol 进入insert mode。a为【从当前光标下一个字符开始插入】。A为【从所在行最后一个字符处插入】
o与O open above/blow 进入insert mode。o为【在当前行下一行插入新的一行】，O为【在当前行上一行插入新的一行】
r与R replace mode 进入replace mode。r为【替换光标处一个字符一次】，R为【替换光标处字符，直至按下Esc健】

eol：end of line。

进入insert mode后会在左下角显示INSERT或REPLACE字样。

insert mode –> command mode

Esc键。

command mode

移动光标
|command|full name|illustration|
|—|—|—|
|h|← |光标左移一个字符|
|j|↓ |光标下移一个字符|
|k|↑ |光标上移一个字符|
|l|→ |光标右移一个字符|

可与数字组合。如，10j为光标下移10个字符。

command full name illustration
ctrl+f page up 向上翻页
ctrl+b page down 向下翻页
ctrl+d 向上翻半页
ctrl+u 向下翻半页
+ next line 光标移动到下一个非空行
- prev line 光标移动到上一个非空行
number+space 即数字+空格。如20(space)光标会移动到后面20个字符的距离
0 “hard” bol 光标移动到行首
$ eol 光标移动到行尾
H screen top 光标移动到屏幕最上方的第一个字符
M screen min’l 光标移动到屏幕中间一行的第一个字符
L screen bottom 光标移动到屏幕最下方的第一个字符
G eof 光标移动到文件最后一行
number+G goto In 跳转到文件的第number行,如20G为跳转到第20行
gg top of file 移动到文件的第一行，相当于1G
number+Enter 光标向下移动数字行，如20(Enter)为光标下20行

bol：beginning of line.

eof：end of file.
